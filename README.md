# 🎵 MOSH_FE


## 🛠 기술 스택

| 분류 | 기술 |
|:---:|:---:|
| 프레임워크 | Next.js 16 |
| 언어 | TypeScript |
| 스타일링 | Tailwind CSS |
| 상태관리 | Zustand |
| 패키지 매니저 | pnpm |

---

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```


---

## 📝 커밋 컨벤션

| 태그 | 설명 |
|:---:|:---|
| `feat` | 새로운 기능 추가 |
| `fix` | 버그 수정 |
| `docs` | 문서 수정 |
| `style` | 코드 스타일 변경 (포매팅, 세미콜론 등) |
| `design` | UI 디자인 변경 (CSS 등) |
| `test` | 테스트 코드 추가/수정 |
| `refactor` | 리팩토링 |
| `build` | 빌드 파일 수정 |
| `ci` | CI 설정 파일 수정 |
| `perf` | 성능 개선 |
| `chore` | 자잘한 수정이나 빌드 업데이트 |
| `rename` | 파일/폴더명 수정 |
| `remove` | 파일 삭제 |

### 커밋 예시

```
feat: 로그인 기능 구현
fix: 버튼 클릭 안되는 버그 수정
docs: README 업데이트
```
